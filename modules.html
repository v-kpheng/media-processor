setVonageMetadata

  • setVonageMetadata(metadata: VonageMetadata): void

  • Sets some metadata for telemetry.

    example

        appId: 'vonage-media-processor-example',
        sourceType: 'test'
      };

      Specifies the addional information being sent with the telemetry collected by the library. If metadata is either undefined or not set the library will not collect/send any telemetry.
